> React 是一个用于构建用户界面（UI）的 JavaScript 库

> Create React App 通过一行命令，即可构建现代化的 Web 应用。

```
npx create-react-app my-app
```

> Next.js 是一个流行的、基于 React 构建的轻量级框架，用于构建静态和服务器端渲染的应用程序。

```
npx create-next-app@latest
```
